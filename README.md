# sort-track-cpp

基于匈牙利匹配和卡尔曼滤波的SORT多目标跟踪算法。



SORT (object tracking) and detector implemented in C++.

```bash
$ mkdir build && cd build
$ cmake ..
$ make
$ cd ..
$ ./sort_cpp_demo
```
